# 企画フェーズ テンプレート集

## questions.md

```markdown
# 企画段階の質問

### Q1: （質問内容）
**回答**: （ユーザーの回答）

### Q2: （質問内容）
**回答**: （ユーザーの回答）
```

## user-stories.md

```markdown
# ユーザーストーリー

## US-1: （ストーリー名）

**ユーザー**: （誰が）
**目的**: （何のために）
**操作**: （何をする）
**優先度**: Must / Should / Could
**受け入れ基準**:
- [ ] （条件1）
- [ ] （条件2）

## US-2: （ストーリー名）

...
```

**記載ガイドライン:**
- 優先度は MoSCoW法（Must / Should / Could / Won't）で分類する
- Won't は user-stories.md には記載しない（questions.md のスコープ外で言及済み）
- 受け入れ基準はユーザー視点で記述する（実装の詳細は含めない）
